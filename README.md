## üéØ What is LEAP?

**ServiceNow LEAP** is the future of enterprise work. A complete UX reimagining that replaces all ServiceNow forms and lists with an AI-powered, people-centric interface.

A living, breathing command center where you orchestrate teams of AI agents, human collaborators, and your own work in a unified, Zen-like stream.

### Key Features
- **Agent-First**: AI agents are first-class citizens
- **Multimodal**: Signals, Decisions, and Activities flow together
- **Card-Based**: Everything is a card (no rows, no tables)
- **Flow-Optimized**: Designed to induce and maintain flow state
- **Universal**: Works across industries, roles, and team structures

## üöÄ Quick Start (5 Minutes)

### Prerequisites
- Node.js 16+ installed
- Firebase account (already configured)
- Modern browser (Chrome/Edge recommended)

### Setup
```bash
# 1. Navigate to project
cd /Users/matthew.phipps/Documents/Antigravity/LEAP

# 2. Install dependencies
npm install

# 3. Start Vite dev server
npm run dev

# 4. Open browser
# Navigate to http://localhost:8003
```

Use test credentials from `SECRETS.md`:

- Password: See SECRETS.md
